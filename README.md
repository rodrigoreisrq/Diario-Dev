# Programa de Doações em Java

Este projeto é um exercício simples em Java que simula um programa de doações.  
O objetivo é praticar conceitos básicos da linguagem, como entrada de dados, condicionais e saída no console.

---

## 📌 Funcionalidades

- Exibe mensagens de boas-vindas e opções de doação.
- Permite ao usuário escolher um valor de doação digitando no console.
- Reconhece valores pré-definidos (1, 5 ou 10 reais).
- Aceita qualquer outro valor e agradece pela contribuição personalizada.

---

## 🛠️ Tecnologias Utilizadas

- **Java** (linguagem de programação)
- **Scanner** (classe para leitura de entrada do usuário)

---

## 🚀 Como Executar

1. Certifique-se de ter o **Java JDK** instalado em sua máquina.
2. Compile o programa:
   ```bash
   javac Main.java
